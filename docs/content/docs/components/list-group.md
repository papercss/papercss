### Simple list group example

<ul class="list-group">
    <li>
        List item #1
    </li>
    <li>
        List item #2
    </li>
    <li>
        List item #3
    </li>
</ul>

### Code

```html
<ul class="list-group">
    <li>List item #1</li>
    <li>List item #2</li>
    <li>List item #3</li>
</ul>
```
